# v0.3 GitHub Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Composite Action that detects drift in generated brief files on every PR, plus the supporting determinism change and docs.

**Architecture:** Remove the in-file timestamp so generated output is byte-deterministic; add a composite action under `.github/actions/repo-brief-check/` that runs `npx @rogerdigital/repo-brief brief` into a temp dir and `git diff --no-index` against committed brief files; add a consumer workflow under `.github/workflows/brief-check.yml`.

**Tech Stack:** TypeScript (ESM, `tsc`), Node ≥ 20, `node:test`, GitHub Actions (composite + workflow), bash for drift-diff step.

**Reference spec:** `docs/superpowers/specs/2026-06-15-v0.3-github-action-design.md`

### Task 4: Create the Composite Action

**Files:**
- Create: `.github/actions/repo-brief-check/action.yml`

- [ ] **Step 1: Create the action directory and `action.yml`**

Create `.github/actions/repo-brief-check/action.yml`:

```yaml
name: RepoBrief Check
description: >-
  Detect drift in generated brief files (AGENTS.md, docs/agent/*.md).
  Fails the job if any committed brief file differs from freshly generated content.

inputs:
  package-version:
    description: "npm version or tag of @rogerdigital/repo-brief to use"
    required: false
    default: "latest"

runs:
  using: composite
  steps:
    - name: Setup Node.js
      uses: actions/setup-node@v4
      with:
        node-version: 20

    - name: Generate fresh brief into temp dir
      shell: bash
      run: |
        npx -y @rogerdigital/repo-brief@${{ inputs.package-version }} brief --cwd "$RUNNER_TEMP/gen"

    - name: Detect drift against committed files
      shell: bash
      run: |
        set -euo pipefail
        gen="$RUNNER_TEMP/gen"
        drift_files=()
        brief_paths=("AGENTS.md" "docs/agent/repo-map.md" "docs/agent/commands.md" "docs/agent/risk-zones.md")

        for rel in "${brief_paths[@]}"; do
          committed="$GITHUB_WORKSPACE/$rel"
          generated="$gen/$rel"
          if [[ ! -f "$committed" ]]; then
            continue
          fi
          if ! git diff --no-index --quiet "$committed" "$generated"; then
            drift_files+=("$rel")
            echo "::group::Drift in $rel"
            git diff --no-index "$committed" "$generated" || true
            echo "::endgroup::"
          fi
        done

        if (( ${#drift_files[@]} > 0 )); then
          echo "::error::Brief files are out of date. Run \`repo-brief brief\` locally and commit the changes."
          printf '  - %s\n' "${drift_files[@]}"
          exit 1
        fi

        echo "All committed brief files are up to date."
```

Run: `node -e "require('node:fs').readFileSync('.github/actions/repo-brief-check/action.yml','utf8'); const yaml = require('node:fs').readFileSync('.github/actions/repo-brief-check/action.yml','utf8'); console.log('action.yml OK, bytes:', yaml.length)"`

  pull_request:
    paths:
      - '**'

permissions:
  contents: read

concurrency:
  group: brief-check-${{ github.ref }}
  cancel-in-progress: true

jobs:
  check:
    name: Brief drift check
    runs-on: ubuntu-latest
    timeout-minutes: 5

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: RepoBrief drift check
        uses: ./.github/actions/repo-brief-check
        with:
